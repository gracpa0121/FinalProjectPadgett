# FinalProjectPadgett

#Data
Post-COVID Video Games Worldwide (2021-2025) https://www.kaggle.com/datasets/otegbolamarvellous/post-covid-video-games-worldwide-2021-2025


#Proposal
In this project, I used my dataset to show if critic and users generally agree on their video game rankings and if games with higher popularity tend to disagree with the critic ratings. I achieved this by making histograms and scatter plots to show the relationships between the variables.

#Report
In conclusion, critic and user scores for video games post COVID are generally aligned and normally positively correlated. The popularity does seem to influence the degree of agreement when it comes to less popular games.A limitation of this dataset was the missing values for the user scores which took out a fair amount of games. Also, just by review count may not show how well the game actually did. In the future, I’d like to do more work with looking at genre, platforms, and release year to better understand what drives the different scores.
